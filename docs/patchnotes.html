Patch Notes

Version 2.0. Changes include:

  1. HTML structure has been simplified and converted, where possible, to semantic HTML.
  2. Style guide created and CSS updated and simplified in line with the new style guide.
  3. Converted all non-manuscript images containing text to HTML so as to be compatible with screenreaders and copyable by users.
  4. Brand new homepage with major design changes, such as a smaller, responsive header, a logo, larger font size and line height.
  5. Expanded the Demonstratives section of Strong Masculine Nouns and added a grammar test.
  6. Increased the number of questions in the grammar test of each Cases module.
  7. Added the ability to load different quizzes of varying complexity levels on the same page. See example here.
  8. Updated grammar tables so they are properly responsive on mobile.
  9. Added togglable info-box component to allow the addition of information without increasing visual complexity. See example here.
  10. Added abbreviation tags to complex words so they can be hovered for explanatory tooltips.
  11. Added audio to the phonetics table to demonstrate correct pronunciation.
  12. Created the Advanced Pronunciation guide to explain more complex aspects of the phonology.
  13. Added the Weak Verbs, Strong Neuter and Feminine Nouns, Pronouns, and Adjectives modules.
  14. Updated About Page to reflect completion of Masters and transition onto the development of the project as a stand-alone learning resource.
  15. Created the timeline for a general overview of the development of Old English.
  16. Created this patch notes page to track changes.
  17. Created the license page for licensing and copyright info.
